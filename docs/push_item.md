## 推文項目

![Image of Push Item](../v1/images/push_item.png) 

1. ### 作者
推文者 ID

2. ### 樓層與推文數
由上往下計算的推文數量，分開計算推、噓與箭頭數

3. ### 推文時間
推文發送的時間

4. ### 推文內容
推文內容，送出後無法編輯

5. ### 推文圖示
共有七種圖示，推、噓、箭頭、原Po、我的推文、好鄉民、超級好友

6. ### 封鎖推文
被封鎖的使用者發送的推文，點擊後可以暫時解除封鎖，長按後選項與推文選項相同，可隨時解除封鎖

7. ### 封鎖推文預覽
被封鎖的使用者發送的推文內包含的預覽項目，點擊後可以暫時解除封鎖
